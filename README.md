Sistema de Base de Datos - FORO

Descripción del Proyecto

Este proyecto consiste en el diseño, modelado e implementación de una base de datos en SQL Server, desarrollado a partir de un conjunto de historias de usuario. Incluye la estructura completa de la base de datos, carga de datos, consultas y documentación del proceso.

El objetivo es representar de manera eficiente la información de Torneos FIFA, garantizando integridad, consistencia y facilidad de consulta. Buenas prácticas de instrucciones en SQL

Objetivos
Diseñar un modelo de datos relacional basado en requerimientos reales
Implementar la base de datos mediante scripts SQL (DDL y DML)
Realizar consultas que respondan a necesidades del negocio
Documentar el proceso de desarrollo de la base de datos

Tecnologías y materiales utilizados:

SQL Server
SQL (DDL y DML)
Excel (historias de usuario)
PDF Informe
JPG Diagrama relacional SQL
